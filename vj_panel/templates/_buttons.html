Buttons

.btn is a class that can be applied to anchors or to buttons. Each .btn must contain [span]Content[/span][i][/i]. This works great in everything except ie6, the solution is to use JS to convert all [button .btn] to anchors and then fake the form action.

The JS to achieve this is included in application.js.

.btn is a class that can be applied to anchors or to buttons. Each .btn must contain [span]Content[/span][i][/i]. This works great in everything except ie6, the solution is to use JS measure the dimensions of the [span] and [i] and then apply relative positioning inline.

The JS to achieve this is included in application.js.
